1.1 #functions.apply-aspect apply-aspect($dims, $value)

Given a set of 2x numbers ($dims) and a value ($value), function applies the aspect ratio to the value and returns the result. This function may also be invoked as applyAspect()

Usage

// Input
div {
    width: apply-aspect( 3 4 );
    height: apply-aspect( 2 3, 600px );
}

// Output
div {
    width: 75%;
    height: 400px;
}
Parameters:
  • $dims
    Space-delimited set of dimensions. Function expects the first number to be the width, second to be the height.
  • $value
    Value against which the aspect ratio will be applied. Defaults to '100%' if no argument is provided.
    Defaults to: 100%

1.2 #functions.get get(@map, $key, $fallback)

Function is a wrapper around the native SASS map-get function.

Usage

// Input
div {
    display: get( $demo-map, display );
}

// Output
div {
    display: block;
}


// NOTE:
// Example above assumes that `$demo-map` incudes a key of 'display', which in turn holds a value of 'block'.

// Input
div {
    overflow: get( $demo-map, overflow, hidden );
}

// Output
div {
    overflow: hidden;
}


// NOTE:
// Example above assumes that `$demo-map` DOES NOT include a key of 'display'. In this case, the fallback value ('hidden') is returned.
Parameters:
  • $map
    Target SASS map.
  • $key
    Target map key.
  • $fallback
    Value to return if target key does not yield a value.

1.3 #functions.get-aspect get-aspect(@dims, $return-height-val)

Given a set of 2x numbers ($dims), function returns their relationship as either an integer or floating point number. This function may also be invoked as getAspect()

Usage

// Input
div {
    width: 100%;
    height: get-aspect( 16 9, false )*100%;
}

// Output
div {
    width: 100%;
    height: 56.25%;
}

// Input
span:before {
    content: "500 is #{get-aspect( 500 200 )} times greater than 200."
}

// Output
span:before {
    content: "500 is 2.5 times greater than 200.";
}
Parameters:
  • $dims
    Space-delimited set of dimensions. Function expects the first number to be the width, second to be the height.
  • $return-height-val
    If set to false, function will return HEIGHT/WIDTH.
    Defaults to: true

1.4 #functions.get-color get-color($color-name, $submap)

Function fetches the specified color from a 'set' of colors, defined on $color-map. This function may also be invoked as getColor().

NOTE: This function may also be used to fetch data from $colorMap.

Usage

// Input
p {
    color: get-color( black );
}

// Output
p {
    color: #000;
}


// NOTE:
// Example above assumes that $color-map includes the key 'black', set to a value of #000.

// Input
h1 {
    background-color: get-color( base, placeholder );
}

// Output
h1 {
    background-color: #FAFAFA;
}

// NOTE:
// Example above assumes that $color-map includes the $submap 'placeholder',
// which in turn has a key called 'base' set to value of #FAFAFA.
Parameters:
  • $color-name
    The color to fetch.
  • $submap
    The set/group of colors which contains $color-name.
    Defaults to: default

1.5 #functions.get-font get-font($font-name, $submap)

Function fetches the specified font from a 'set' of fonts, defined on $font-map. This function may also be invoked as getFont().

NOTE: This function may also be used to fetch data from $fontMap.

Usage

// Input
body {
    font-family: get-font( copy );
}

// Output
body {
    font-family: "Times New Roman", serif;
}


// NOTE:
// Example above assumes that $font-map includes the $submap 'default',
// which in turn has a key called 'copy' set to a value of "'Times New Roman', serif".

// Input
h1 {
    font-family: get-font( primary, display );
}

// Output
h1 {
    font-family: "Helvetica", "Arial", sans-serif;
}

// NOTE:
// Example above assumes that $font-map includes the $submap 'display',
// which in turn has a key called 'primary' set to a value of "'Helvetica', 'Arial', serif".
Parameters:
  • $font-name
    The font to fetch.
  • $submap
    The set/group of fonts which contains $font-name.
    Defaults to: default
