1.1 #functions.apply-aspect apply-aspect($dims, $value)

Given a set of 2x numbers ($dims) and a value ($value), function applies the aspect ratio to the value and returns the result. This function may also be invoked as applyAspect()

Usage

// Input
div {
    width: apply-aspect( 3 4 );
    height: apply-aspect( 2 3, 600px );
}

// Output
div {
    width: 75%;
    height: 400px;
}
Parameters:
  • $dims
    Space-delimited set of dimensions. Function expects the first number to be the width, second to be the height.
  • $value
    Value against which the aspect ratio will be applied. Defaults to '100%' if no argument is provided.
    Defaults to: 100%
